EMS Media System Help Guide

EMS Media System is a schedule-based presentation tool. You build a service or event schedule in the main window, preview or cue items locally, then send the live output to your selected display with the Present button.

Getting Started

Main Layout

  • The left side of the Media tab is your schedule and settings area
  • The main preview area shows the currently selected item, cued item, or Bible workspace
  • Use Add Media or drag and drop files into the app to build your schedule
  • The header Present button starts the selected scheduled item on the chosen output display

Schedule

Building and Running a Schedule

  • The schedule supports video, audio, images, PowerPoint files, and Bible items
  • Click a scheduled item to load it into preview when nothing is live
  • Click a different item during a live presentation to cue it without interrupting the live output
  • Drag scheduled items to reorder them
  • Use the Auto/Stop button on each row to control whether the next item advances automatically
  • Use the X button to remove one item, or Clear to empty the whole schedule

Preview and Cueing

  • Use the custom controls under the preview to play, pause, scrub, loop, and change volume
  • For audio and video cues, scrubbing can set a custom cue start time for that scheduled item
  • Schedule rows show Live for the item currently on output and Start @ when a cue start time has been saved
  • Audio-only items play locally in the control window while still acting as the live presentation

Bible Workspace

Creating Scripture Slides

  • Click the Bible button in the sidebar to open the scripture workspace
  • Choose a Bible version, type a reference, and select verses from the chapter list
  • Use the live preview to adjust font, size, text color, background color, and optional image or video background
  • Show Now sends the current Bible text straight to the presentation display
  • Add to Schedule inserts the current Bible item into the schedule for later use
  • Click Bible verse text to quickly show it now, add the selection, or schedule each selected verse separately
  • The Apply buttons can push your Bible style choices across all Bible items in the current project

Presentation Workflow

  1. Add files or Bible items to the schedule
  2. Choose the target screen in Settings under Output Display
  3. Leave Autoplay enabled if you want items to begin playing as soon as they are taken live
  4. Select or cue the item you want to start with
  5. Click Present to send it to the selected display
  6. While live, click another scheduled item to stage it, then take it live when ready
  7. Click Stop or press Esc to end the presentation
Tip: When possible, EMS switches the live output in place so the presentation window stays open while the next scheduled item loads.

Projects and Files

Saving Your Work

  • Use Open, Save Project, and Save Project As from the menu to work with .emproj files
  • Projects remember your schedule order, cue points, Bible items, preview selection, and project text styling
  • EMS also keeps an autosaved project state and restores it when possible on the next launch

Working vs Portable Projects

  • Save Project writes a working project that references files in their current locations
  • Export Portable Project creates a packed project intended to travel with its media
  • If files have been moved, scheduled items are marked missing and can be repaired with Relink Missing Files

Streams Tab

Network Playback

  • Use the Streams tab for YouTube or direct stream URLs
  • Streams are separate from the Media schedule workflow
  • When a stream is presented, it opens on the selected display using the same presentation window system

Keyboard Shortcuts

  • F1 - Open this help window
  • F5 - Start presentation
  • Esc - Stop the active presentation
  • Ctrl/Cmd + O - Open project
  • Ctrl/Cmd + S - Save project
  • Ctrl/Cmd + Shift + S - Save project as
  • Ctrl/Cmd + Q - Close the main window
